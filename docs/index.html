Ollama Curriculums

De un PDF a un resumen profesional generado por inteligencia artificial local — sin enviar datos a la nube.

↓

Tecnologías

Python 3 pypdf markdownify requests Ollama qwen2.5:3b-instruct argparse

El proyecto combina procesamiento de documentos con inteligencia artificial local. Todo se ejecuta en la máquina del usuario — privacidad total.

Pipeline de procesamiento

1

PDF → Texto

pypdf extrae el texto de cada página del currículum. Se muestra el progreso página a página.

2

Texto → Markdown

markdownify convierte el texto plano a formato Markdown estructurado y legible.

3

Markdown → Ollama

El CV en Markdown se envía al modelo qwen2.5:3b-instruct a través de la API local de Ollama en localhost:11434.

4

Resumen profesional

La IA genera un resumen en tercera persona y emite una valoración sobre la idoneidad del candidato para el puesto docente.

Código destacado

Extracción de PDF con progreso:

reader = PdfReader(path_pdf)
for i, page in enumerate(reader.pages, start=1):
    text = page.extract_text() or ""
    full_text += text + "\n\n"
    print(f"Página {i}/{len(reader.pages)} extraída")

Comunicación con Ollama:

url = f"{host}/api/generate"
payload = {
    "model": modelo.strip(),
    "prompt": prompt,
    "stream": False,
}
response = requests.post(url, json=payload, timeout=600)
return response.json()["response"].strip()

Prompt de análisis

El prompt instruye al modelo para actuar como experto en selección de personal:

"Eres un experto en selección de personal y redacción de perfiles profesionales. Recibirás un currículum en formato Markdown. Tu tarea: leer el CV cuidadosamente, identificar habilidades clave, tecnologías, experiencia relevante y logros. Escribir un resumen profesional conciso en tercera persona (7–10 líneas, ~200 palabras). Emite una opinión acerca de si el perfil es válido para el puesto de trabajo: profesor de ciclos formativos."

Mejoras aplicadas

  • Docstrings con formato Google en pdf_to_md().
  • Verificación de archivo con os.path.isfile() antes de procesarlo.
  • Progreso por página al extraer el PDF.
  • argparse para poder pasar archivo y modelo por línea de comandos.
  • Errores específicos: ConnectionError, Timeout, HTTPError separados.
  • Timing: muestra cuántos segundos tardó la IA en responder.
